# detect

Browser/OS detections

## Install

```shell
yarn add @a1motion/detect
```

## Usage

```javascript
import detect from "@a1motion/detect";

detect = {
  ie: boolean;
  safari: boolean;
  firefox: boolean;
  opera: boolean;
  chrome: boolean;
  mac: boolean;
  win: boolean;
  ios: boolean;
  android: boolean;
  linux: boolean;
  desktop: boolean;
  mobile: boolean;
}
```
